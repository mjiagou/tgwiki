# TGwiki 项目记忆

## 项目概述
- **名称**: TGwiki - Telegram知识库
- **网站**: https://wiki.tgnav.org/
- **技术栈**: VuePress 2.0 + vuepress-theme-hope + pnpm
- **所属**: TGNAV

## 内容结构
- 侧边栏5大分类：热门专题、电报入门、基础操作、进阶玩法、账号与安全
- 文章分类标签：基础教程、进阶教程、科普知识
- 专题子目录：topics/ton/、topics/bot/、topics/game/、topics/resource/
- 文章总数：85+ 篇 Markdown

## 文章写作规范
- Frontmatter: title, shortTitle, description, icon, category, tag
- Head meta: keywords 用于SEO
- 图片路径: https://cdn.jsdelivr.net/gh/tgwiki/images/tfa/
- 内部链接格式: [文本](./filename.html)
- 使用VuePress自定义容器: ::: tip, ::: warning, ::: danger, ::: info, ::: details, ::: caution
