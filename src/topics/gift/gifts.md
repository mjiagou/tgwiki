# Telegram 礼物系统完全指南：发送、收藏与 NFT 转赠

Telegram 礼物系统（Gifts）是 Telegram 推出的一项社交互动功能，允许用户在聊天中向好友发送虚拟礼物。部分限量礼物还可以升级为基于 TON 区块链的 NFT，在链上自由交易和转让。

本指南将带你全面了解礼物系统的运作方式。

---

## 一、什么是 Telegram 礼物？

### 1.1 功能简介

Telegram 礼物是一种可在私聊中发送的虚拟物品，类似于"动态表情+收藏品"的结合体。收到礼物后，礼物会展示在聊天界面中，同时保存在接收者的**个人资料礼物墙**上。

### 1.2 礼物的核心特征

| 特征 | 说明 |
|------|------|
| **可视化展示** | 礼物以精美的动画效果出现在聊天中 |
| **个人资料展示** | 收到的礼物会展示在你的个人主页上 |
| **限量发行** | 部分礼物有发行数量上限，具有收藏价值 |
| **NFT 升级** | 限量礼物可升级为链上 NFT，支持转让和交易 |
| **Telegram Stars 支付** | 使用 Telegram Stars（星星）购买发送 |

### 1.3 与 Premium 的关系

- 所有用户都可以**接收**礼物
- 发送礼物需要消耗 **Telegram Stars**（星星）
- Premium 会员在购买礼物时可享受**折扣优惠**

::: tip 💡 小提示
Telegram Stars 可通过应用内购买获得，也可通过 Fragment 平台使用 TON 代币购买，后者通常更划算。
:::

## 二、礼物的类型

### 2.1 普通礼物

普通礼物无发行数量限制，任何人都可以随时购买发送。价格通常在 25~100 Stars 之间。

常见类型包括：
- 🎂 生日蛋糕
- 🌹 玫瑰花束
- 🎉 派对礼炮
- 🧸 泰迪熊
- 🏆 奖杯

### 2.2 限量礼物

限量礼物有**总发行量上限**，售完即止。发行量从几百到数万不等。限量礼物的核心特点：

- 每件限量礼物都有唯一的**编号**（如 #001/1000）
- 编号越小，通常越受收藏者追捧
- 售完后无法再从官方渠道购买
- 可在二级市场（Fragment）上交易

### 2.3 限量礼物的稀缺层级

| 层级 | 发行量 | 稀有度 |
|------|--------|--------|
| 超稀有 | < 500 | 🔥 极高 |
| 稀有 | 500 ~ 2,000 | ⭐ 高 |
| 普通 | 2,000 ~ 10,000 | 📦 中 |
| 大众 | > 10,000 | 📋 低 |

::: warning ⚠️ 注意
限量礼物售完后，二级市场价格可能远高于原价，也可能低于原价。礼物价值受市场供需影响，请理性消费。
:::

## 三、如何发送礼物

### 3.1 发送步骤

1. 打开与好友的**私聊**界面
2. 点击输入框左侧的 **📎 附件按钮**（或 **😊 表情按钮**）
3. 选择 **Gift**（礼物）选项
4. 浏览可选礼物列表，选择心仪的礼物
5. 添加可选的**祝福语**
6. 点击 **Send Gift** 确认发送
7. 系统扣除相应数量的 Telegram Stars

### 3.2 发送限制

- **仅限私聊**：目前不支持在群组或频道中发送礼物
- **好友关系**：部分设置下，需要对方未屏蔽你
- **Stars 余额**：账户内需有足够的 Stars 余额
- **地区限制**：礼物功能在部分地区可能暂不可用

::: details 📱 不同客户端入口
- **iOS / Android**：聊天界面 → 附件/表情按钮 → Gift
- **Desktop**：聊天界面 → 右下角表情按钮 → Gift 标签
- **macOS**：与 Desktop 类似，部分旧版本可能暂不支持
:::

## 四、礼物展示与个人资料墙

### 4.1 个人资料礼物墙

收到的礼物会展示在你的个人主页上，其他用户点击你的头像即可看到：

- **最近收到**：展示最近收到的礼物（最多 6 个）
- **全部礼物**：点击展开查看完整列表
- **礼物详情**：点击单个礼物可查看发送者、时间、祝福语

### 4.2 隐藏与展示设置

你可以控制礼物在个人资料上的展示方式：

1. 进入 **设置 → 隐私设置**
2. 找到 **Gifts**（礼物展示）选项
3. 可选择：
   - **所有人可见**：任何人都能看到你的礼物墙
   - **仅联系人可见**：只有好友能查看
   - **隐藏所有礼物**：不在个人资料展示任何礼物

::: tip 🎁 隐藏 ≠ 删除
隐藏礼物只是不在个人资料上展示，礼物本身仍保留在你的账户中。你可以随时重新展示。
:::

## 五、限量礼物升级为 NFT

这是礼物系统最核心的进阶功能——将限量礼物转化为链上资产。

### 5.1 升级条件

- 礼物必须是**限量版**（有发行总量限制）
- 升级需要支付一定的 **TON 手续费**（通常 1~5 TON）
- 升级后礼物变为 **NFT**，可在 TON 区块链上验证

### 5.2 升级步骤

1. 进入 **设置 → Gifts**（或个人资料 → 礼物墙）
2. 找到你想升级的限量礼物
3. 点击礼物 → 选择 **Upgrade to NFT**
4. 系统会显示手续费金额，确认后扣除 TON
5. 升级完成后，礼物获得**链上唯一标识**
6. 此时可以将其**转让**或**挂售**

### 5.3 升级后发生什么？

| 升级前 | 升级后 |
|--------|--------|
| 存在于 Telegram 账户内 | 存在于 TON 区块链上 |
| 不可转让 | 可自由转让和交易 |
| 仅 Telegram 内可见 | 在 Fragment 等平台可见 |
| 无链上所有权证明 | 有 NFT 所有权证明 |

::: warning ⚠️ 升级后不可逆
NFT 升级是**不可逆**操作。升级后礼物将变为链上资产，无法再恢复为普通 Telegram 礼物状态。请确认后再操作。
:::

## 六、NFT 礼物的转让与交易

### 6.1 转让给其他用户

升级为 NFT 后，你可以将礼物直接转赠给其他 Telegram 用户：

1. 在礼物墙中找到已升级的 NFT 礼物
2. 点击 **Transfer**（转让）
3. 选择接收者（从联系人中选）
4. 确认转让——需支付少量 TON 作为链上 Gas 费
5. 对方接收后，NFT 所有权转移完成

### 6.2 在 Fragment 上挂售

Fragment 是 Telegram 官方的 NFT 交易平台，支持礼物 NFT 的买卖：

1. 访问 [fragment.com](https://fragment.com)
2. 连接你的 **TON 钱包**（推荐 Tonkeeper）
3. 进入 **Gifts** 板块
4. 选择你要出售的 NFT 礼物
5. 设置售价（以 TON 计价）
6. 确认挂单，支付 Gas 费
7. 买家购买后，TON 自动转入你的钱包

### 6.3 购买他人出售的礼物

1. 在 Fragment 的 Gifts 市场浏览在售礼物
2. 可按**类型、编号、价格**筛选
3. 找到心仪的礼物后点击购买
4. 支付 TON，NFT 转入你的钱包
5. 可选择将 NFT 导入回 Telegram 展示

::: tip 💡 编号溢价
编号为 #1 的礼物通常具有最高溢价（被称为 "Number 1"）。小编号（#2~#10）也有较高溢价。购买时注意编号差异。
:::

## 七、Telegram Stars 获取方式

发送礼物需要 Stars，获取 Stars 的主要途径：

### 7.1 应用内购买

直接在 Telegram 内购买，支持 Apple Pay / Google Pay / 信用卡：

| 数量 | 参考价格（美元） |
|------|-----------------|
| 100 Stars | $1.99 |
| 500 Stars | $7.99 |
| 1,000 Stars | $14.99 |
| 2,500 Stars | $34.99 |
| 5,000 Stars | $64.99 |

::: warning ⚠️ 平台差异
通过 App Store / Google Play 购买需支付 30% 平台抽成。通过 Telegram Desktop 或 Web 版购买可避开抽成，价格更优。
:::

### 7.2 Fragment 购买

在 [fragment.com](https://fragment.com) 上使用 TON 代币购买 Stars，通常比应用内购买更划算：

1. 访问 Fragment → 选择 **Buy Stars**
2. 输入你的 Telegram 账号
3. 支付 TON
4. Stars 自动充入你的 Telegram 账户

## 八、常见问题

### Q1：普通用户能收到礼物吗？

可以。所有 Telegram 用户都能**接收**礼物，不要求是 Premium 会员。但发送礼物需要消耗 Stars。

### Q2：礼物会过期吗？

不会。收到的礼物永久保存在你的账户中，不会过期或消失。

### Q3：普通礼物能升级为 NFT 吗？

不能。只有**限量礼物**支持 NFT 升级。普通礼物无发行量限制，不具备 NFT 属性。

### Q4：升级为 NFT 后还能在 Telegram 里展示吗？

可以。升级后的 NFT 礼物仍然可以在个人资料墙上展示，同时它也是链上可验证的资产。

### Q5：可以退款吗？

已发送的礼物**不可退款**。购买 Stars 后，Stars 也不可退回为现金。但 NFT 礼物可以在 Fragment 上出售变现。

### Q6：发送礼物时对方会知道是我吗？

会。礼物发送时会附带你的身份信息和祝福语，接收者可以在礼物详情中查看发送者。

### Q7：礼物有手续费吗？

- 发送礼物：消耗 Stars，无额外手续费
- 升级 NFT：需支付 TON 链上 Gas 费
- Fragment 交易：平台收取少量佣金

## 九、安全与防骗提醒

::: danger 🚨 重要安全提示
1. **官方渠道购买**：只在 Telegram 应用内或 Fragment 官网购买礼物和 Stars，不要相信第三方代购。
2. **警惕低价骗局**：有人声称"低价出售限量礼物"，通常是骗局。NFT 交易请通过 Fragment 官方平台进行。
3. **不要泄露助记词**：升级和交易 NFT 需要使用 TON 钱包，任何人索要助记词都是骗子。
4. **理性消费**：限量礼物有收藏价值但也有市场风险，不要盲目追高。
5. **核实交易地址**：在 Fragment 交易时，确认网址为 `fragment.com`，防止钓鱼网站。
:::

## 十、总结

Telegram 礼物系统将社交互动与区块链收藏巧妙结合：

| 功能 | 普通用户 | Premium 会员 |
|------|---------|-------------|
| 接收礼物 | ✅ | ✅ |
| 发送礼物 | ✅（需 Stars） | ✅（享折扣） |
| 展示礼物墙 | ✅ | ✅ |
| 升级 NFT | ✅（需 TON） | ✅（需 TON） |
| 转让 NFT | ✅ | ✅ |
| Fragment 交易 | ✅ | ✅ |

对于普通用户，礼物是增进友谊的社交工具；对于收藏者，限量礼物 NFT 是新兴的数字藏品赛道。无论哪种角色，都建议先了解机制再参与。
