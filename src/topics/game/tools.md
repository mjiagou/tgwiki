# Telegram 实用小程序推荐：无需安装App，在电报里收邮件、查快递

## 1. Gmail Bot (官方)
**ID**: `@GmailBot`

这是 Google 官方开发的机器人。绑定你的 Gmail 账号后，你可以在 Telegram 里直接接收新邮件通知，甚至可以直接回复邮件。

* **优点**：无需一直挂着梯子收邮件，Telegram 会即时推送。
* **安全**：官方认证，无需担心隐私泄露。

<!-- ![Gmail机器人收信界面](/assets/images/game/gmail-bot.jpg) -->

## 2. Tracking (快递查询)
**ID**: `@TrackingBot`

海淘党神器！支持全球 400+ 快递公司的物流查询（包括 DHL, UPS, FedEx, 中国邮政等）。

* **用法**：直接发送快递单号给它，它会自动识别并推送物流状态更新。
* **技巧**：可以给单号备注名称，例如“iPhone - SF123456”。

## 3. File Converter (文件转换)
**ID**: `@newfileconverterbot`

手机上没有格式转换软件？发给它就行。
* **功能**：支持图片转 PDF、视频转 MP3、Webp 转 JPG 等几十种格式互转。
* **限制**：免费版有文件大小限制，但处理文档绰绰有余。

## 4. IFTTT
**ID**: `@IFTTT`

自动化神器。可以将 Telegram 连接到你的 Twitter, Instagram, 智能家居等。
* **场景**：当我在 Instagram 发照片时，自动同步发到 Telegram 频道；每天早上 8 点在群里自动发送天气预报。
