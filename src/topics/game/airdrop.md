# Telegram 游戏空投(Airdrop)新手指南：如何通过玩游戏赚取加密货币？

## 什么是“点击赚钱” (Tap-to-Earn)？

自 2024 年 Notcoin 爆火以来，Telegram 上涌现了大量 Tap-to-Earn 游戏。
**核心逻辑**：
1.  **前期**：项目方为了获取用户，开发简单的小游戏。
2.  **中期**：用户通过玩游戏、邀请好友赚取“游戏积分”。
3.  **后期 (TGE)**：项目方发行加密货币 (Token)，并将 Token 按比例“空投”给持有游戏积分的用户。用户可以在交易所卖出变现。

## 核心步骤：如何接收空投？

想要拿到钱，你必须有一个钱包。

1.  **准备钱包**：推荐使用 **Tonkeeper**（[点击查看教程](/topics/ton/ton.html)）。
2.  **绑定钱包**：在游戏界面找到 "AirDrop" 或 "Wallet" 选项，点击 "Connect Wallet"。
3.  **完成任务**：通常需要关注项目方的 Twitter、Telegram 频道，并保持每日活跃。
4.  **等待快照**：项目方会在某一天宣布“快照”，记录所有人的积分。
5.  **领取 (Claim)**：Token 上线后，你需要支付少量的 TON 作为手续费（Gas费）来领取代币。

## 🚨 风险警示：防骗指南

空投圈鱼龙混杂，请务必遵守以下原则：

1.  **不给私钥**：任何游戏要求你输入助记词或私钥，**100% 是骗子**。
2.  **不乱签名**：绑定钱包时，仔细看弹窗。如果显示 `Send Transaction` 且金额很大，立刻拒绝。正常的绑定操作是不扣钱的。
3.  **零撸为主**：对于不明背景的项目，不要充值任何资金。

::: tip 准备一点点 TON
虽然领空投是免费的，但最后领取（Claim）和转账时，你需要钱包里有大约 0.5 ~ 1 个 TON 用来支付区块链网络费（Gas）。可以参考 [如何充值钱包](/topics/ton/wallet.html)。
:::
