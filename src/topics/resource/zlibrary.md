# Z-Library 回归？教你使用 Telegram 机器人免费下载电子书 (最新)

## Z-Library 官方机器人还在吗？

众所周知，Z-Library 的网页端经常被封锁，访问非常困难。但它的 **Telegram 机器人** 一直非常稳定，是目前下载电子书体验最好的方式。

**核心优势**：
* **无需翻墙**：只要你能连上 Telegram，就能下书。
* **格式全**：支持 EPUB, PDF, MOBI 等多种格式。
* **直接推书**：文件直接发送到你的聊天窗口，不用跳转网盘。

<!-- ![Z-Library机器人对话界面](/assets/images/resource/zlib-bot.jpg) -->

## 第一步：绑定个人账号 (关键)

现在的 Z-Library 不再提供公共机器人（因为容易被封），每个人需要绑定自己的专属机器人。

1.  访问 Z-Library 官方登录页（寻找可用的镜像站，如 `singlelogin.re`）。
2.  登录你的账号（如果没有就注册一个）。
3.  点击页面右下角的 **Link to Telegram bot** (连接到电报机器人)。
4.  网页会跳转到 Telegram，并自动打开一个类似 `@ZLibrary12345Bot` 的机器人。
5.  点击 **Start**。

## 第二步：如何找书下载

绑定成功后，这个机器人就是你的私人图书馆了。

1.  **直接发送书名**：例如发送 `三体`。
2.  机器人会回复搜索结果列表。
3.  **选择书籍**：点击你想下载的那本书下方的命令（例如 `/download_123`）。
4.  **接收文件**：等待几秒钟，机器人会把电子书文件直接发给你。
5.  **转发收藏**：建议把文件转发到“收藏夹 (Saved Messages)”，永久保存。

## 替代方案：如果不绑定账号？

如果你觉得注册账号太麻烦，可以使用一些第三方的搜书机器人（来源于 Z-Library 数据库）：

* **搜书机器人**: `@sobooksbot`
* **BookDown**: `@bookdownbot`

::: warning 注意
第三方机器人通常有每日下载限制（例如每天只能下 3 本），且广告较多。长期阅读建议还是绑定官方 Z-Library 账号。
:::
