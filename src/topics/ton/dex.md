# TON链DEX交易入门：STON.fi / DeDust 操作全攻略与安全避坑

如果你已经有了 Tonkeeper 钱包，也充值了一些 TON，那么接下来大概率会想：**怎么用 TON 买其他代币？** 答案就是：**DEX（去中心化交易所）**。

本文将带你从零开始，一步步学会在 TON 链上使用 DEX 进行交易，同时帮你避开新手最容易踩的坑。

---

## 一、什么是 DEX？为什么用它？

### 1.1 DEX vs CEX：有什么区别？

| 对比项 | DEX（去中心化交易所） | CEX（中心化交易所） |
|:---|:---|:---|
| **代表** | STON.fi、DeDust、Uniswap | 币安、欧易 OKX、Coinbase |
| **资金托管** | 你自己保管私钥 | 交易所保管你的币 |
| **注册要求** | 不需要注册，连钱包就能用 | 需要注册 + KYC 实名认证 |
| **上币门槛** | 任何人都可以发币上线 | 项目方付费上币，审核严格 |
| **币种数量** | 非常多，新项目也能交易 | 相对较少，以主流币为主 |
| **手续费** | 链上 Gas 费 + 交易手续费 | 交易手续费 + 提币费 |
| **安全性** | 智能合约风险，但不会跑路 | 平台风险，可能被盗或冻结 |
| **适合人群** | 追求自由、想玩新项目 | 追求安全、交易主流币 |

::: tip 一句话总结
**CEX 就像银行，DEX 就像现金交易。**
银行安全方便，但要实名、要审核；现金交易自由匿名，但要自己看好钱。
:::

### 1.2 为什么要在 TON 链上用 DEX？

**三个主要原因：**

1. **玩 Telegram 小游戏空投变现**
   - 很多 Tap-to-Earn 游戏（如 Notcoin、Catizen）的代币会先上 DEX
   - 空投的币可以直接在 DEX 上卖掉换成 TON 或 USDT

2. **投资早期新项目**
   - TON 生态的新项目通常先上 DEX，后上 CEX
   - 看好某个项目可以早期买入，等上交易所可能已经涨了很多

3. **匿名交易**
   - 不需要 KYC，连钱包就能交易
   - 适合注重隐私的用户

::: warning ⚠️ 风险提示
DEX 上的币**没有审核**，空气币、骗局币非常多。
新手建议只交易已经上了大交易所的主流币，不要乱买不知名的小币。
**投资有风险，入市需谨慎！**
:::

## 二、TON 生态两大头部 DEX

TON 链上有两大主流 DEX，占据了绝大部分交易量：

| 对比项 | STON.fi | DeDust |
|:---|:---|:---|
| **地位** | TON 最大的 DEX，交易量第一 | TON 第二大 DEX |
| **官网** | ston.fi | dedust.io |
| **代币** | STON | DUST |
| **界面** | 简洁直观，新手友好 | 功能更丰富，适合进阶 |
| **流动性** | 最好，主流币深度足 | 次好，有些小币深度更好 |
| **特色功能** | 发射台（Launchpad）、质押 | 限价单、聚合交易 |
| **推荐** | ⭐⭐⭐⭐⭐ 新手首选 | ⭐⭐⭐⭐ 补充使用 |

::: tip 建议
新手推荐先用 **STON.fi**，界面简单，操作容易上手。
等熟悉了以后，可以两个都用，哪边价格好就在哪边交易。
:::

## 三、交易前的准备

### 3.1 你需要准备什么？

1. **一个去中心化钱包**
   - 推荐：[Tonkeeper](./ton.md)（最主流、最好用）
   - 其他：Tonhub、MyTonWallet 等也可以

2. **一些 TON 币**
   - 用来支付 Gas 费（交易手续费）
   - 以及作为交易的本金
   - 建议至少准备 **2-5 TON**（1 TON 用来付 Gas，剩下的用来交易）

3. **获取 TON 的方法**
   - 从交易所（币安、欧易）提币到钱包
   - 用 Telegram 内置钱包转账
   - P2P 购买

### 3.2 连接钱包

以 **STON.fi + Tonkeeper** 为例：

1. 打开 Tonkeeper App
2. 点击底部的 **浏览器**（Browser）标签
3. 在搜索栏输入 `ston.fi` 并打开
4. 点击页面右上角的 **Connect Wallet**（连接钱包）
5. 选择 **Tonkeeper**
6. 在弹出的确认窗口中点击 **Connect**（连接）
7. 连接成功后，右上角会显示你的钱包地址余额

::: tip 小技巧
也可以直接在 Telegram 里打开 DEX 的 Telegram Mini App 版本，
比如 STON.fi 的机器人是 `@stonfi_bot`，体验更流畅。
:::

## 四、STON.fi 交易保姆级教程

### 4.1 买入代币（用 TON 换其他币）

**以用 TON 买 NOT（Notcoin）为例：**

1. 打开 STON.fi，确认钱包已连接
2. 在交易界面，**From**（支付）选择 **TON**
3. **To**（收到）选择 **NOT**（或者你想买的其他币）
   - 如果找不到币，可以在搜索框输入代币名称或合约地址
4. 输入你想花多少 TON（比如 10 TON）
5. 系统会自动计算你能收到多少 NOT

::: danger 🚨 买币前必看！
**1. 核对合约地址！**
   - 很多骗子会发同名的假币
   - 一定要从项目官方渠道确认合约地址
   - 名字一样不代表是真币！

**2. 检查流动性！**
   - 流动性太低的币不要买
   - 流动性池少于 1000 TON 的要小心
   - 流动性低 = 滑点大 = 可能卖不出去
:::

6. 检查以下参数：
   - **Price（价格）**：当前兑换价格
   - **Price Impact（价格影响）**：你的交易会造成多大滑点，越小越好
   - **Minimum Received（最少收到）**：滑点范围内你最少能拿到多少
   - **Fee（手续费）**：交易手续费，通常 0.3% 左右 + 少量 Gas 费

7. 确认无误后，点击 **Swap**（兑换）
8. Tonkeeper 会弹出确认窗口，检查信息后点击 **Confirm**（确认）
9. 等待几秒，交易完成！你会收到新的代币

### 4.2 卖出代币（用其他币换 TON/USDT）

卖出操作和买入基本一样，只是方向反过来：

1. **From**（支付）选择你想卖出的币（比如 NOT）
2. **To**（收到）选择 **TON** 或 **USDT**
3. 输入卖出数量
4. 检查价格、滑点、手续费
5. 点击 **Swap** → **Confirm** 确认
6. 等待交易上链完成

::: tip 滑点怎么设？
- 交易量大的主流币（TON/USDT）：滑点设 **0.5% - 1%** 就够了
- 交易量小的小币：滑点可能需要设 **3% - 5%** 甚至更高
- 滑点设太小会导致交易失败，但 Gas 费照扣
- 滑点设太大可能被"三明治攻击"，以很差的价格成交
- **新手建议用默认值，不要乱改**
:::

### 4.3 什么是滑点？为什么会有滑点？

**滑点（Slippage）** 是指你下单时的预期价格，和实际成交价格之间的差距。

**为什么会有滑点？**

举个通俗的例子：
> 你想去水果店买 100 斤苹果。
> 店里标价 5 块钱一斤，但店里只有 50 斤 5 块的，
> 剩下 50 斤是 6 块的存货。
> 结果你买 100 斤的平均价格就变成了 5.5 块，
> 这多出来的 0.5 块就是滑点。

**影响滑点的因素：**
- 流动性池越大 → 滑点越小
- 你的交易量越小 → 滑点越小
- 交易越频繁 → 滑点变化越快

## 五、进阶概念科普

### 5.1 流动性池（Liquidity Pool）

**流动性池**就是 DEX 里的"资金池"。

传统交易所用"订单簿"（买一卖一），而 DEX 用的是"流动性池"模型（AMM 自动做市商）。

**简单理解：**
- 有人把两种币按比例存进池子（比如 TON + USDT）
- 这些人叫 **流动性提供者（LP）**
- 他们赚的是**交易手续费的分成**
- 你交易的时候，就是和这个池子做交换

### 5.2 无常损失（Impermanent Loss）

::: tip 新手可以暂时跳过
如果你只是做交易（买币卖币），不需要了解无常损失。
只有当你想**提供流动性（做 LP）赚手续费时**，才需要关注这个概念。
:::

**无常损失**是指你提供流动性后，币价波动带来的损失。

简单来说：
- 如果你把币放在流动性池里
- 币价涨了，你赚的比单纯持有少
- 币价跌了，你亏的比单纯持有多
- 只有币价不怎么变的时候，你才能稳稳赚手续费

**结论：** 提供流动性不一定比持有更划算，新手不建议随便做 LP。

### 5.3 价格影响（Price Impact）

就是你的交易对币价的影响程度。

- 交易量小 + 池子大 → 价格影响小（好）
- 交易量大 + 池子小 → 价格影响大（差）

一般来说，价格影响低于 **1%** 是比较理想的。

## 六、安全避坑指南（新手必读！）

### 6.1 防骗指南：99% 的人都踩过这些坑

#### 🚫 坑 1：买了假币（同名骗局）

**骗法：** 骗子发行一个和热门币同名的代币，让你以为是真的。

**防范方法：**
1. **只从官方渠道获取合约地址**（项目官网、官方推特、官方群）
2. 不要相信私信发给你的"优质币"
3. 搜币的时候看清楚，名字一样不代表是同一种币
4. 检查流动性和交易量，太低的大概率是假的

#### 🚫 坑 2：授权骗局（Approve 钓鱼）

**骗法：** 你点了某个假网站的"授权"按钮，骗子就能转走你钱包里所有的币。

**防范方法：**
1. **永远不要在陌生网站授权你的钱包！**
2. 授权前看清楚授权的是什么币、授权给谁
3. 只在你信任的大平台（STON.fi、DeDust 等）上操作
4. 定期去钱包里查看和取消不必要的授权

#### 🚫 坑 3：私钥泄露

**骗法：** 各种方式骗你交出助记词/私钥。

**防范方法：**
1. 助记词永远不要告诉任何人
2. 不要截图、不要存手机里、不要发微信
3. 官方客服不会问你要助记词
4. 群管不会私信你"帮忙解决问题"

#### 🚫 坑 4：高收益骗局（土狗/rug pull）

**骗法：** 包装一个看起来很厉害的项目，吸引你买币，然后项目方卷钱跑路。

**防范方法：**
1. 项目信息不透明的不要投
2. 团队匿名的要小心
3. 承诺高收益的基本都是骗局
4. 新手只玩已经上了大交易所的主流币

### 6.2 交易安全好习惯

✅ **小额测试**：第一次交易某个币，先用 1-2 TON 试试水
✅ **自己做研究**：不要听别人喊单就冲进去
✅ **分散投资**：不要把所有钱都砸在一个币上
✅ **设置止损**：亏到一定程度就割肉，不要死扛
✅ **定期提现**：赚到钱了及时转去冷钱包或变现
✅ **保护私钥**：这是你的生命线，比什么都重要

## 七、常见问题 FAQ

### Q1: 交易失败了，为什么 Gas 费还是扣了？

因为交易虽然失败了，但矿工（验证者）已经帮你把交易打包上链了，所以 Gas 费还是要付的。

**常见失败原因：**
- 滑点设太低
- 钱包里 TON 不够付 Gas
- 网络拥堵

### Q2: 为什么同一个币，两个 DEX 价格不一样？

这是正常的。DEX 的价格是由流动性池里的比例决定的，不同池子的价格会有细微差异。

专业的交易员会用"套利"来赚取差价，但对新手来说，哪边划算就在哪边交易就行。

### Q3: 交易后多久到账？

TON 链速度很快，通常 **5-10 秒**就能确认到账。
如果网络拥堵，可能需要等几十秒到几分钟。

### Q4: 怎么查看我的交易记录？

在 Tonkeeper 钱包里：
- 点击底部的 **钱包** (Wallet)
- 往下翻就能看到所有交易记录
- 点击某一笔可以查看详细信息和 TxID

也可以在区块链浏览器里查看，比如：
- `tonscan.org`
- `ton.sh`
- `explorer.toncoin.org`

### Q5: DEX 里的"池子（Pool）"是什么？

就是流动性池。可以理解为交易对的"库存"。
池子越深（钱越多），交易滑点越小，体验越好。

### Q6: 我可以把币提到交易所吗？

可以。只要那个交易所支持该币种的 TON 链充提。

**注意：**
- 从钱包提币到交易所，通常需要填 Memo/Tag
- 不填 Memo 币可能会丢失！
- 具体参考：[转账没到账怎么办？](./trans-fail.md)

## 八、总结与建议

**给新手的 5 条建议：**

1. **先用 STON.fi** —— 界面友好，不容易出错
2. **只交易主流币** —— 别乱买不知名的小币
3. **小额试水** —— 先拿一点钱练手，熟悉操作
4. **安全第一** —— 保护好助记词，别乱点链接
5. **不懂就别碰** —— 搞不懂的东西不要硬上

::: tip 最后想说
DEX 是加密世界的双刃剑，它给你自由，也把风险交给了你自己。
在享受自由交易的同时，一定要对自己的资产负责。
多学习、少冲动、不贪婪，才能在这个市场活得久。
:::
