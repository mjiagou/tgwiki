# Tonkeeper 钱包完全指南：安装、创建、安全备份与 Telegram 生态交互

Tonkeeper 是 TON 区块链上最受欢迎的**去中心化钱包**，由第三方团队开发，不依赖 Telegram 官方。如果你想玩 Telegram 小游戏领空投、参与 TON 生态 DeFi、购买 NFT，Tonkeeper 是必备工具。

本文将全面讲解 Tonkeeper 的安装、创建、使用和安全注意事项。

---

## 一、为什么需要 Tonkeeper

### 1.1 内置钱包的局限性

Telegram 内置钱包（@Wallet）虽然方便，但有以下局限：

| 局限 | 影响 |
|:---|:---|
| **托管式** | 私钥由官方保管，账号被封可能丢币 |
| **需要 KYC** | 大额交易需要实名认证 |
| **功能有限** | 无法连接游戏、DeFi、NFT 平台 |
| **隐私较低** | 所有交易都和 Telegram 账号绑定 |

### 1.2 Tonkeeper 的优势

Tonkeeper 是**非托管钱包**，核心优势：

1. **私钥自持**：只有你拥有 24 个助记词，官方无法动用你的资产
2. **无需 KYC**：不需要任何实名认证，完全匿名
3. **生态入口**：可以连接 Fragment、STON.fi、Getgems 等 TON 生态应用
4. **玩游戏必备**：Telegram 热门游戏（Notcoin、Hamster Kombat 等）领空投必须用 Tonkeeper
5. **多平台支持**：iOS、Android、浏览器扩展

### 1.3 什么时候用 Tonkeeper vs 内置钱包

| 场景 | 推荐钱包 |
|:---|:---:|
| 发红包、小额转账 | 内置钱包 |
| P2P 买卖加密货币 | 内置钱包 |
| 玩 Telegram 游戏领空投 | Tonkeeper |
| 参与 TON DeFi | Tonkeeper |
| 购买 NFT/匿名号码 | Tonkeeper |
| 大额资产存储 | Tonkeeper |
| 需要匿名隐私 | Tonkeeper |

::: tip 建议
两个钱包配合使用：
- **内置钱包**：日常小额支付、P2P 交易
- **Tonkeeper**：玩游戏、DeFi、大额资产存储
:::

## 三、创建钱包与安全备份（生命线！）

### 3.1 创建新钱包步骤

**步骤 1：打开 App**

打开 Tonkeeper，选择 **Create new wallet**（创建新钱包）

**步骤 2：查看助记词**

App 会生成 **24 个英文单词**，这就是你的**助记词（Seed Phrase）**。

::: danger 🚨 绝对安全红线
这 24 个单词是**你资产的唯一凭证**！

**绝对不能做的事：**
1. ❌ **不要截图**：手机相册可能被恶意 App 读取
2. ❌ **不要复制到剪贴板**：剪贴板可能被监控
3. ❌ **不要发给任何人**：任何人要助记词都是骗子
4. ❌ **不要存到云端**：网盘、邮箱都不安全
5. ❌ **不要用输入法打出来**：输入法可能上传数据

**正确做法：**
- 用**纸和笔**，按顺序抄写下来
- 检查每个单词拼写正确
- 存放在保险柜、书里或其他安全地方
- 可以做 2 份备份，放在不同地点
:::

**步骤 3：验证助记词**

App 会要求你按顺序点击几个单词，确保你抄写正确。

**步骤 4：设置 PIN 码**

设置一个 4-6 位 PIN 码，开启 FaceID/TouchID。

**完成！** 你的去中心化钱包创建成功了。

### 3.2 助记词安全原则

| 安全等级 | 存储方式 | 风险 |
|:---|:---|:---|
| ⭐⭐⭐⭐⭐ | 纸质抄写，存保险柜 | 最低 |
| ⭐⭐⭐⭐ | 金属刻字板（防火防水） | 很低 |
| ⭐⭐⭐ | 多处纸质备份 | 低 |
| ⭐⭐ | 加密 PDF 存 U 盘 | 中等 |
| ⭐ | 截图存手机 | 高 |
| ❌ | 存网盘/邮箱/聊天记录 | 极高（必丢） |

### 3.3 导入已有钱包

如果你之前有 Tonkeeper 钱包，想在新设备恢复：

1. 打开 Tonkeeper，选择 **Import wallet**（导入钱包）
2. 按顺序输入你的 24 个助记词
3. 设置 PIN 码
4. 钱包恢复成功

## 四、钱包界面与功能

### 4.1 主界面说明

打开 Tonkeeper 后，你会看到：

| 区域 | 功能 |
|:---|:---|
| **资产余额** | 显示你的 TON 和其他代币余额 |
| **接收按钮** | Receive - 查看钱包地址，用于收款 |
| **发送按钮** | Send - 发送代币给其他地址 |
| **交易记录** | 查看所有交易历史 |
| **浏览功能** | Browser - 打开 TON 生态应用 |

### 4.2 支持的代币

Tonkeeper 支持所有 TON 链上的代币：

- **TON**（Toncoin）：TON 链原生代币
- **USDT**：泰达币（TON 链版本）
- **Jettons**：TON 链上的各种代币
- **NFT**：TON 链上的 NFT 藏品

### 4.3 查看钱包地址

1. 点击 **Receive**
2. 你会看到：
   - **钱包地址**：一串以 `UQ` 或 `EQ` 开头的字符
   - **二维码**：可扫描接收
3. 点击地址可复制

::: tip 注意
Tonkeeper 充值**不需要填 Memo**，直接用地址即可。
:::

## 五、如何充值 TON

### 5.1 从交易所充值

**步骤：**

1. 在 Tonkeeper 点击 **Receive**，复制钱包地址
2. 打开交易所（币安、欧易、Gate 等）
3. 进入 **提币/提现** 页面
4. 选择 **TON**
5. 选择网络：**TON Network**（注意不是 ERC20 或其他）
6. 粘贴 Tonkeeper 地址
7. 确认提币

**到账时间**：通常 1-5 分钟

### 5.2 从内置钱包转入

如果你想把 Telegram 内置钱包的 TON 转到 Tonkeeper：

1. 打开内置 Wallet
2. 点击 **Withdraw**（提现）
3. 选择 **TON**
4. 粘贴 Tonkeeper 地址
5. 输入金额，确认提现

### 5.3 从其他钱包转入

支持从任何支持 TON 链的钱包转入：

- 其他 TON 钱包
- 交易所账户
- 只要对方支持 TON 链提现即可

## 六、如何发送 TON

### 6.1 发送给其他地址

**步骤：**

1. 打开 Tonkeeper
2. 点击 **Send**
3. 粘贴接收方地址（或扫描二维码）
4. 输入金额
5. 确认交易（需要输入 PIN 或 FaceID）
6. 等待交易确认

**手续费**：TON 链手续费很低，通常 0.01-0.05 TON

### 6.2 发送到交易所

**步骤：**

1. 在交易所获取 TON 充值地址
2. 在 Tonkeeper 点击 Send
3. 粘贴交易所地址
4. 输入金额，确认发送

::: tip 提示
发送到交易所通常**不需要填 Memo**（交易所会自动识别）。
但如果交易所提示需要 Memo，请按要求填写。
:::

## 七、连接 Telegram 小游戏（重要！）

### 7.1 为什么要连接游戏

玩 Telegram 热门游戏（Notcoin、Hamster Kombat、Blum 等），领空投时必须连接 TON 钱包。空投的代币会直接发到你的 Tonkeeper。

### 7.2 连接步骤

**步骤 1：在游戏中选择连接钱包**

1. 打开游戏机器人
2. 找到 **Connect Wallet** 或 **Wallet** 选项
3. 选择 **Tonkeeper**

**步骤 2：授权连接**

1. 系统会自动唤起 Tonkeeper App
2. 查看请求内容
3. 点击 **Connect Wallet** 确认

::: warning 安全提醒
连接钱包时：
- ✅ 只是"连接"，不需要输入密码或支付费用
- ❌ 如果弹出要求支付 TON，立刻拒绝
- ❌ 如果要求输入助记词，100% 是骗子，关闭页面
:::

**步骤 3：连接成功**

返回游戏，钱包地址会显示已连接状态。

### 7.3 领取空投

当游戏代币上线（TGE）时：

1. 在游戏中找到 **Claim** 或 **领取** 选项
2. 确认领取到 Tonkeeper
3. 需要支付少量 TON 作为 Gas 费
4. 代币到账后，可以在 Tonkeeper 查看

### 7.4 常见游戏连接示例

| 游戏 | 连接方式 |
|:---|:---|
| **Notcoin** | 游戏内 → Wallet → Tonkeeper |
| **Hamster Kombat** | 设置 → Connect Wallet → Tonkeeper |
| **Blum** | Wallet → Connect → Tonkeeper |
| **Catizen** | 钱包图标 → Tonkeeper |

## 八、连接 DEX 与 DeFi

### 8.1 什么是 DEX

DEX（去中心化交易所）让你可以直接用 TON 交换其他代币，不需要注册交易所账户。

TON 生态主要 DEX：
- **STON.fi**：最大的 TON DEX
- **DeDust**：另一主流 DEX

### 8.2 连接 DEX 步骤

**方法一：通过 Tonkeeper 浏览器**

1. 打开 Tonkeeper
2. 点击 **Browser**（浏览）
3. 选择 STON.fi 或其他 DEX
4. 钱包自动连接

**方法二：通过网页**

1. 用手机浏览器打开 [ston.fi](https://ston.fi)
2. 点击 **Connect Wallet**
3. 选择 Tonkeeper
4. 自动唤起 App 授权

### 8.3 交易步骤

以 STON.fi 为例：

1. 连接钱包后，选择要交易的代币对
2. 输入交易金额
3. 查看汇率和滑点
4. 点击 **Swap**（交换）
5. Tonkeeper 弹出确认窗口
6. 确认交易，等待完成

::: tip 更多 DEX 教程
详见 [DEX 交易入门指南](./dex.html)。
:::

## 九、连接 Fragment 购买匿名号码

Fragment 是 Telegram 官方的 NFT 平台，可以购买：

- **收藏用户名**：如 @john 这类用户名 NFT

1. 每次使用后确认 App 已锁定
2. 定期检查交易记录
3. 只连接可信的网站和应用
4. 大额资产考虑使用硬件钱包

**❌ 不应该做的事：**
1. 不要在公共 WiFi 下操作钱包
2. 不要点击陌生链接连接钱包
3. 不要相信任何"官方客服"要助记词
4. 不要在越狱/Root 的手机上使用

### 10.3 防骗指南

| 骗局 | 套路 | 应对 |
|:---|:---|:---|
| **假客服** | 说你账户异常，要助记词 | 拉黑举报 |
| **假空投** | 发链接让你连接钱包领空投 | 不点陌生链接 |
| **钓鱼网站** | 伪装成 STON.fi 等官网 | 检查 URL 拼写 |
| **授权骗局** | 连接钱包时要求转账签名 | 正常连接不收钱 |

### 10.4 资产安全建议

- **小额日常**：Tonkeeper 够用
- **大额资产**：考虑硬件钱包（如 Ledger 配合 Tonkeeper）
- **定期备份**：检查助记词是否安全存放
- **分散存储**：不要把所有资产放在一个钱包

## 十一、常见问题 FAQ

### Q1: Tonkeeper 收手续费吗？

Tonkeeper 本身不收手续费，但每笔交易需要支付 **TON 链 Gas 费**（通常 0.01-0.05 TON）。

### Q2: 充值需要填 Memo 吗？

不需要。Tonkeeper 是非托管钱包，充值只需要地址，不需要 Memo。

### Q3: 助记词丢了怎么办？

**无法找回！** 助记词是资产的唯一凭证，丢失 = 永久丢失资产。
请务必安全备份！

### Q4: 可以在多个设备使用同一个钱包吗？

可以。在任何设备安装 Tonkeeper，用助记词导入即可恢复同一个钱包。

### Q5: Tonkeeper 支持哪些币？

支持所有 TON 链上的代币：
- TON（Toncoin）
- USDT（TON 链版本）
- 各种 Jetton 代币
- NFT

### Q6: 忘记 PIN 码怎么办？

重新导入钱包即可：
1. 卸载重装 Tonkeeper
2. 选择 Import wallet
3. 输入助记词
4. 重新设置 PIN 码

### Q7: 和 Telegram 内置钱包有什么区别？

| Tonkeeper | 内置钱包 |
|:---|:---|
| 去中心化，自己保管私钥 | 托管式，官方保管私钥 |
| 不需要 KYC | 需要 KYC |
| 可连接游戏/DeFi | 功能有限 |
| 账号被封不影响资产 | 账号被封可能丢币 |
| 充值不需要 Memo | TON 充值需要 Memo |

### Q8: 发送到错误地址能追回吗？

**不能！** 区块链交易不可逆。发送前请仔细核对地址。

### Q9: 为什么连接游戏时要我签名？

正常情况下，连接钱包只是"授权"，不收钱。
- 如果是"连接"请求，确认即可
- 如果是"转账"请求（显示要发送 TON），拒绝它

### Q10: Tonkeeper 官方能冻结我的账户吗？

不能。Tonkeeper 是去中心化钱包，官方没有权限动你的资产。只有拥有助记词的人才能操作钱包。
