## 关于电报宝典

电报宝典（TGwiki）是一个专注于 Telegram（电报/纸飞机）的中文知识库。我们致力于为中文用户提供高质量、系统化的 Telegram 使用指南，从基础的下载安装、账号注册，到高级的隐私设置、机器人使用，帮助每一位用户充分发挥 Telegram 的强大功能。

Telegram 以其高度加密、无审查、跨平台同步等特性受到全球用户的喜爱。然而，由于语言障碍和功能复杂性，许多中文用户难以充分利用 Telegram 的全部潜力。我们的使命就是降低使用门槛，让更多人享受这款优秀通讯工具带来的便利。

## 快速解决常见问题

刚开始使用 Telegram？以下是最常见的问题和解决方案，帮助你快速入门：

- **汉化中文界面**：Telegram 原生不支持中文？安装中文语言包即可汉化界面。[查看教程](./language.html)

- **收不到验证码**：注册或登录时验证码迟迟不到？可能是网络或运营商问题。[解决方法](./notcomesms.html)

- **无法私聊陌生人**：新账号被限制无法主动私聊？了解原因及解除方法。[解除限制](./spam.html)

- **账号被封禁**：登录提示号码被封禁？不要慌张，按步骤申诉。[解封指南](./banned.html)

- **无法查看敏感内容**：部分频道提示无法显示？调整隐私设置即可。[解除限制](./pornios.html)

- **隐私安全设置**：保护你的账号安全和隐私，新手必做的几项设置。[隐私指南](./privacy.html)
