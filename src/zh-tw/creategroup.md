# Telegram 群組 (Group) 建立與設定完全指南 (超級群組)

如果您希望建立群組時直接升級為「超級群組」（擁有管理員權限、記錄公開等功能），建議在建立時直接邀請一個機器人進群。

1. 複製一個 Bot 的使用者名稱：`@keyworddel_bot` 或 `@UserMeta_Bot` 或 `@AntiServiceMessage_Bot`
2. 點擊 `建立群組` -> 在添加成員時，**貼上**剛剛複製的使用者名稱並邀請它。

## 「建立群組」按鈕在哪裡？

- **iOS App**：`聊天` 分頁 -> 點擊右上角最右邊的 `撰寫圖示` -> 選擇 `建立群組`。
- **Android App**：點擊 `左上角三條線` (漢堡選單) -> 選擇 `建立群組`。
- **Desktop 電腦版**：點擊 `左上角三條線` -> 選擇 `建立群組`。
- **macOS 電腦版**：點擊 `主介面搜尋框後方圖示` -> 選擇 `建立群組`。
