# Telegram 封存 (Archive) 功能詳解：如何隱藏不常用的聊天室

封存聊天（Archive）的功能允許您隱藏目前在「聊天」分頁上的對話，將不常用的群組和頻道放入「已封存的聊天」中，並可以在以後有需要時隨時查看。您可以封存個別聊天或群組，以更好的方式整理與組織您的聊天列表。這類似於 LINE 的「隱藏」或 QQ 的「群助手」功能。

## 手機版

- **iOS**：在聊天列表中，對著想封存的對話**向左滑動**，即可選擇封存。
- **Android**：在`聊天列表` -> `長按對話` -> 點擊右上角的`封存圖示`（一個向下箭頭的盒子）；封存後，聊天列表頂部會顯示一欄「已封存的聊天」。

### 管理已封存的內容

- **取消封存**：對著已封存的對話向左滑動，即可取消封存（還原到主列表）。
- **隱藏封存欄位**：對著頂部的「已封存的聊天」欄位向左滑動，可以將其隱藏。
- **找回封存欄位**：在聊天列表**向下拉**，即可再次顯示「已封存的聊天」欄位；若要將其固定在頂部，可再對其向左滑動並選擇置頂。

## 電腦版 (Desktop)

- **操作方法**：右鍵點擊任何聊天室 -> 選擇 `封存` (Archive)。
- **查看封存**：封存的聊天室會出現在列表的最頂端（如果有的話），或者可以透過左上角的選單查看。
