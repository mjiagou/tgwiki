# Telegram 頻道 (Channel) 建立教學：從 0 到 1 打造你的專屬頻道

頻道類似於 Facebook 粉絲專頁或微信公眾號，這是一種單向廣播的工具。只有**管理員**才能發布訊息，成員（訂閱者）只能接收並閱讀訊息，無法在裡面直接聊天（除非開啟了留言功能）。

## 「建立頻道」按鈕在哪裡？

- **iOS App**：點擊底部 `聊天` 分頁 -> 點擊右上角最右邊的 `撰寫圖示` (紙筆圖案) -> 選擇 `建立頻道`。
- **Android App**：點擊主介面右下角的 `筆形圖示` -> 選擇 `建立頻道`。
- **Desktop 電腦版**：點擊左上角的 `漢堡選單` (三條線) -> 選擇 `建立頻道`。
- **macOS 電腦版**：點擊主介面搜尋框後方的 `撰寫圖示` -> 選擇 `建立頻道`。
