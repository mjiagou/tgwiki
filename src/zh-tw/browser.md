# Telegram 內建瀏覽器功能詳解：連結預覽、TON 網站支援與快取設定

在新版 Telegram 中，新增了內建瀏覽器功能，用戶可以直接使用內建瀏覽器在 Telegram 內開啟任何連結，而無需跳轉到外部瀏覽器（如 Chrome 或 Safari）。

## 主要功能

- **網頁搜尋**：直接在瀏覽器內搜尋內容。
- **加入收藏**：將常用網頁加入書籤。
- **即時預覽**：可以將支援的頁面轉化為即時預覽（Instant View）格式，閱讀更流暢。
- **Web3 支援**：直接開啟 TON 區塊鏈網站。
- **外部開啟**：您可以隨時點擊右上角按鈕，選擇在外部瀏覽器中開啟當前網頁。

## 設定選項

您可以透過內建瀏覽器頁面右上角的 `設定` 按鈕，或進入 `設定` -> `外觀` (或聊天設定) 中向下滑動找到 `App 內瀏覽器` 選項，修改相關設定。

::: details 操作演示

![Telegram 內建瀏覽器功能](https://cdn.jsdelivr.net/gh/tgwiki/images/tfa/browser2.jpg)

:::

可以設定的項目包含：

- **開啟/關閉 App 內建瀏覽器**：若關閉，連結將直接透過外部瀏覽器開啟。
- **清除 Cookies**：刪除網站存儲的登入狀態等資訊。
- **清除快取 (Cache)**：釋放瀏覽器佔用的儲存空間。
- **清空歷史記錄**：刪除瀏覽紀錄。
- **例外網站**：設定特定網站「不使用」內建瀏覽器開啟。
- **修改預設搜尋引擎**：選擇您喜歡的搜尋引擎（如 Google, DuckDuckGo 等）。
