# 如何聯絡 Telegram 官方客服？電報官方支援信箱與 Twitter 帳號總整理

## 官方支援管道

- **App 內建客服**：`App` -> `設定` -> `提問` (Ask a Question) -> `Telegram 志工支援`。
- **支援頁面**：[https://telegram.org/support](https://telegram.org/support)（適合網頁端填寫表單）
- **FAQ 常見問答**：[https://telegram.org/faq](https://telegram.org/faq)
- **API 文件**：[https://core.telegram.org](https://core.telegram.org)
- **機器人開發支援**：[@BotSupport](https://t.me/BotSupport)
- **官方 Twitter (X)**：[https://twitter.com/telegram](https://twitter.com/telegram)
- **登入問題專用 Twitter**：[https://twitter.com/smstelegram](https://twitter.com/smstelegram)（收不到簡訊可私訊此帳號）
- **登入常見問題**：[https://telegra.ph/Common-Login-Problems-03-12](https://telegra.ph/Common-Login-Problems-03-12)

## 官方電子信箱

**通用支援信箱**：`support@telegram.org`

::: tip

此信箱現已較少使用，回覆速度較慢，建議優先透過 App 內支援或填寫網頁表單聯絡。

:::

**特定問題專用信箱**：

- **帳號恢復**：`recover@telegram.org`
- **登入相關**：`login@stel.com`
- **安全問題**：`security@telegram.org`
- **濫用檢舉**：`abuse@telegram.org`
- **版權申訴（DMCA）**：`dmca@telegram.org`
- **兒童安全問題**：`childabuse@telegram.org`
- **新聞媒體聯絡**：`press@telegram.org`

## 用戶端反饋群組 (Bug 回報)

- **iOS**：[@tgiostests](https://t.me/tgiostests)
- **macOS**：[@macswift](https://t.me/macswift)
- **Desktop (電腦版)**：[@TelegramDesktopTalk](https://t.me/TelegramDesktopTalk)

- **Telegram 總部地址**：Business Central Towers, Tower A, Office 1003/1004, P.O. Box 501919, Dubai, United Arab Emirates
- **Telegram 支援志工計畫**：Telegram 有一個「支援倡議」（Telegram Support Initiative），招募全球志工回答用戶問題。[點擊查看官方說明](https://tsf.telegram.org/)
