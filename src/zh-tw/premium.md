# Telegram Premium 會員功能詳解與開通儲值教學

由於營運成本考量，Telegram 推出了付費訂閱服務 —— **Telegram Premium**。

## 專屬功能

- **無廣告**：在公開頻道中不再顯示贊助廣告。
- **更快的下載速度**：解除頻寬限制，下載文件速度最大化。
- **語音訊息轉文字**：點擊語音訊息旁的按鈕即可將其轉換為文字。
- **進階對話管理**：可設定預設對話資料夾、自動封存並靜音非聯絡人訊息。
- **動態大頭貼**：個人頭像可以使用動態影片，在列表和對話中都會動。
- **徽章標示**：名字旁邊會顯示一顆專屬的星星徽章。
- **專屬貼圖**：可以使用帶有全螢幕特效的 Premium 專屬貼圖。
- **專屬訊息回應**：解鎖更多 Emoji 用於訊息回應 (Reactions)。
- **專屬 App 圖示**：可更換 Telegram App 在手機桌面上的圖示。
- **自訂 Emoji 包**：可在訊息和說明中使用自訂的 Emoji。
- **使用頻道身分發言**：在群組中可以選擇以自己管理的頻道身分發送訊息。
- **訊息自動翻譯**：即時翻譯整個聊天室的內容。
- **限時動態 (Stories) 優先顯示**：您的動態會排在他人前面，並可設定過期時間、永久查看記錄。
- **助力 (Boost)**：獲得 4 次助力機會，幫助頻道/群組解鎖動態功能。
- **自訂顏色**：自訂名字顏色和訊息氣泡樣式。
- **查看相似頻道**：加入一個頻道時，顯示類似主題的頻道推薦。
- **隱私設定**：可設定僅接收來自「聯絡人」和「Premium 用戶」的私訊。
- **資料夾標籤**：在對話列表中直接顯示資料夾名稱。
- **商務功能**：暫時體驗 [Telegram Business](./business.html) 的全部功能。

## 上限翻倍

- **單一檔案大小限制**：`2000` MB -> `4000` MB (4GB)
- **資料夾上限數**：`10` -> `20`
- **資料夾中的對話數**：`100` -> `200`
- **「全部」列表置頂對話數**：`5` -> `10`
- **資料夾內置頂對話數**：`100` -> `200`
- **加入群組和頻道數**：`500` -> `1000`
- **建立公開群組和頻道數**：`10` -> `20` (公開連結)
- **同時登入帳號數**：`3` -> `4`
- **收藏貼圖數**：`5` -> `10`
- **收藏 GIF 數**：`200` -> `400`
- **個人簡介字數限制**：`70` -> `140`
- **媒體說明 (Caption) 字數限制**：`1024` -> `4096`

## 價格與訂閱方式

### App Store (iOS)

- **台灣**：約 NT$ 100~130/月 (視匯率波動)
- **香港**：HK$ 38.88/月
- **美國**：$4.99/月
- **土耳其**：₺69.99/月
- **訂閱方式**：在 App `設定` -> `Telegram Premium` 訂閱，透過 Apple ID 付款。

### Google Play (Android 商店版)

- **台灣**：約 NT$ 100~130/月
- **香港**：HK$ 39.90/月
- **美國**：$4.99/月
- **土耳其**：₺69.99/月
- **訂閱方式**：在 App `設定` -> `Telegram Premium` 訂閱，透過 Google Play 付款。

### 官方機器人：[@PremiumBot](https://t.me/PremiumBot) (最推薦 🔥)

這是 Telegram 官方提供的直購管道，避開了蘋果/谷歌的抽成，因此價格通常最便宜。

- **美金**：$3.99/月，$28.99/年（約 6 折優惠）
- **特點**：
    - 支付貨幣與手機號碼歸屬地可能有關。
    - **需綁定信用卡** (Visa, MasterCard 等)。
    - 預設開啟**自動續訂**，若要取消請向 Bot 發送 `/stop` 指令。
    - 支付由第三方平台 Smart Glocal 處理。
- **限制**：
    - iOS/Android **商店版** App 無法使用此 Bot 支付。
    - **Android 官網版 (APK直裝)** 或 **電腦版 (Desktop)** 可以使用此 Bot 支付。

## 贈送會員

您可以將 Premium 會員作為禮物贈送給其他用戶。
- 贈送他人會員，在其有效期內您將獲得 `3` 次助力 (Boost) 機會。
- 被贈予者會獲得 `1` 次助力機會。

同時，頻道主也可以透過 **贈禮活動 (Giveaways)** 隨機贈送 Premium 會員給訂閱者，這可以快速提升頻道的助力等級。
