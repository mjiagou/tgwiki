# Telegram 帳號被封鎖 (Banned) 怎麼申訴解除？官方解封範本分享

登入時若提示 `This phone number is banned`（`此手機號碼已被封鎖`），說明此手機號已經被 Telegram 官方封鎖了，既不能登入也無法刪除帳號。

![tfa-banned.jpg](https://cdn.jsdelivr.net/gh/tgwiki/images/tfa/banned.jpg)

:::

## 解除封鎖方法

### 方法一：發送申訴郵件

點擊 `Help/幫助`，使用自己任意一個電子信箱，編輯郵件發送至以下任一地址：

- login@stel.com
- recover@telegram.org
- login@telegram.org
- abuse@telegram.org

**郵件標題：**

```
Banned phone number: +1 xxx xxx xxxx
```

**郵件內容範本：**

```
I'm trying to use my mobile phone number: +1 xxx xxx xxxx
But Telegram says it's banned. Please help.
I need this account. It's on my most used number.

App version: 10.14.5 (49455)
OS version: SDK 34
Device Name: Android

Thank you.
```

::: tip 注意事項

1. 郵件中的 `+886 9xx xxx xxx` 請替換為您被封鎖的 **Telegram 登入手機號碼**。
2. `+886` 是台灣的國際區碼（香港為 `+852`，中國大陸為 `+86`），請務必填寫正確，否則官方無法處理。
3. 郵件寄出後大約需等待 1~7 天，前提是您的違規情況不嚴重。
4. 由於官方客服人力不足，建議可以向上述三個信箱多次發送，以提高被看到的機率。

:::

### 方法二：填寫官方支援表單

用手機或電腦瀏覽器打開 Telegram 官方支援頁面：[https://telegram.org/support](https://telegram.org/support)

在 `Please describe your problem`（請描述您的問題）欄位中填入以下內容（可自行調整）：

```
Dear Sir/Madam,
My number +1 xxx xxx xxxx been banned and i am not able to figure out the reason for supension. Please help me to recover my account.
Thank you.
```

::: tip

同樣請注意，文中的 `+886 9xx xxx xxx` 必須替換為您實際被封鎖的手機號碼。

:::

### 方法三：透過 Twitter 聯絡官方

打開 Telegram 官方 Twitter (X)：[https://twitter.com/smstelegram](https://twitter.com/smstelegram)

直接**私訊 (DM)** 發送方法 1 或方法 2 的申訴內容給官方帳號。

::: tip
有不少群友回報，這個方法還蠻有效的，因此補充上來供大家參考。
:::

### 備註

每個帳號的違規情況不同，有人成功了也有人沒成功，無法一概而論。

如果以上三種方法都嘗試過了，等待一個月後仍未解封，那通常就是無解了（永久封鎖）。

## 如何防止被封鎖？

1. **不要發廣告**：這是最常見的封鎖原因。
2. **少私訊陌生人**：主動私訊（私聊）過多陌生人容易觸發風控。
3. **避免被檢舉**：言行舉止要注意，盡量避免被他人按下「檢舉」按鈕。
4. **謹慎使用接碼平台**：接碼平台的號碼通常是共用的，如果平台出事（如被判定為機器人農場），可能會導致關聯的帳號被連坐封鎖。
5. **謹慎使用 GV 等虛擬號碼**：有些 Google Voice 號碼可能之前已經被人使用過且有不良記錄。
6. **避免同 IP 批量註冊**：在同一個 IP 或網路環境下批量註冊幾百上千個帳號，極高機率會被集體封鎖。
7. **不要頻繁更換登入設備**：保持登入環境穩定。
8. **不要短時間加入大量群組**：新帳號短時間內一次性加入過多頻道或群組，會被系統判定為機器人，導致強制登出甚至封號。
9. **避免濫用伺服器資源**：例如短時間內發送大量大型檔案。
10. **嚴禁違規內容**：涉及暴力、血腥、毒品、政治極端主義、人獸交、兒童色情（CSAM）等，以及其他違反 Telegram 使用條款的內容，一旦被封鎖，基本上**無法解封**。
