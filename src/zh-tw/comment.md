# Telegram 頻道開啟留言功能教學：如何連結討論群組 (Channel Comments)

**開啟方法**：`頻道管理` -> `討論 (Discussion)` -> `連結群組` -> `加入一個群組`

設定完成後，在頻道發送的訊息下方都會顯示「**留言**」按鈕，點擊即可對當前頻道訊息發表留言，留言的內容也會同步顯示在連結的群組中。

::: details 介面截圖

![tfa-comment.jpg](https://cdn.jsdelivr.net/gh/tgwiki/images/tfa/comment.jpg)

:::

::: tip

點擊留言進入的是一個針對該則訊息專屬的討論串（Thread），只會顯示當前訊息的留言內容。

:::

如果在連結的群組中回覆頻道訊息或留言內容，也會同步顯示在頻道訊息的討論串中。

此外，在已連結頻道的群組中，直接回覆訊息也會以被回覆的訊息為基礎建立一個討論串，右鍵/長按/點擊被回覆的訊息即可看到「x 則回覆」，點擊即可進入討論串查看詳情。
