# iOS苹果手机解除Telegram敏感内容限制教程 (开启老司机模式)

Telegram如果你进入某个群或频道遇到如下提示：
```This channel can't be displayed because it was used to spread pornographic content.```

## 具体原因

- 有人在群里发了色情内容，被人举报后，被Telegram官方限制了；
- Telegram官方只会处理举报的，不会主动审查，如果没人举报就不会被限制；
- 只有iOS&Mac上App Store里下载的客户端会有此限制，这是因为Apple的限制，上架App Store的内容限制，其他客户端无此限制，可以正常进入群。

::: important

巴基斯坦、哈萨克斯坦、伊朗、印度、印度尼西亚、马来西亚等地区的号码没有此选项，无法解除敏感内容限制。

:::

## 操作方法

### 方法一

打开Telegram Desktop客户端（适用于Windows/macOS），左上角三短线：`设置`->`外观`->开启`显示18+内容`；

::: details 操作演示

:::

### 方法二

打开Telegram for Android客户端，`设置`->`外观`->打开`显示18+内容`。

::: details 操作演示

![tfa-pornios.jpg](https://cdn.jsdelivr.net/gh/tgwiki/images/tfa/pornios.jpg)

:::

### 方法三

- 手机或电脑浏览器访问 [官方网页版（A版）](https://web.telegram.org/) ；
- 扫描二维码或输入手机号码登录；
- 主界面->左上角三短线->`Settings`->点击`Privacy and Security`->滑动到底部，打开`Show 18+ content`。 

::: warning

必须用浏览器打开，验证码是优先发送到之前已登录的Telegram设备/客户端/App上。

如果你在Telegram直接打开链接，就无法看到Telegram上的验证码了。

:::

::: details 操作演示

![web-pornios1.jpg](https://cdn.jsdelivr.net/gh/tgwiki/images/web/pornios1.jpg)

:::

### 方式四

- 手机或电脑浏览器访问 [官方网页版（K版）](https://web.telegram.org/k/) ；
- 扫描二维码或输入手机号码登录；
- 主界面->左上角三短线->`Settings`->`Privacy and Security`->打开`Disable filtering`。 

::: warning

必须用浏览器打开，验证码是优先发送到之前已登录的Telegram设备/客户端/App上。

如果你在Telegram直接打开链接，就无法看到Telegram上的验证码了。

:::

:::

## 开心使用

通过上面四种方式的任意一种开启后，再通过下面三种方式之一来开始使用Telegram。

- 重启Telegram iOS客户端；
- 等几分钟再重启下客户端；
- 重新登录。
