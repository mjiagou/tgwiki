# Telegram 注销账号完全指南：立即删除与自动销号机制

出于更换手机号、隐私清理或暂时退网等原因，你可能需要注销当前的 Telegram 账号。Telegram 提供了**手动立即注销**与**到期自动销号**两种机制。

需要强调的是，**注销账号是不可逆的行为**。一旦注销，云端所有聊天记录、群组及联系人将永久清空。本文将为您详细介绍注销步骤、防坑避险指南以及销号前后的关键注意事项。

---

## 一、手动立即注销账号（立即生效）

如果你希望立刻注销账号，无法通过 Telegram 手机 App 或电脑客户端直接操作，必须通过官方的**网页端管理后台**完成。

### 1.1 详细操作步骤
1. 使用浏览器打开 Telegram 官方管理后台链接：
   👉 [https://my.telegram.org/auth?to=deactivate](https://my.telegram.org/auth?to=deactivate)
2. **输入手机号**：在 `Your Phone Number` 输入框中输入你的 Telegram 绑定手机号。**必须使用国际格式**（如中国大陆手机号为 `+8613800000000`，切勿漏掉 `+` 号和国别码）。
3. **获取验证码**：点击 `Next`。此时，验证码**不会以手机短信 (SMS) 形式发送**，而是会发送到你已经登录的 Telegram 客户端内（来自官方系统账号 `Telegram` 的服务通知）。
4. **填写验证码**：返回浏览器，将客户端收到的字母和数字混合验证码复制粘贴到 `Confirmation code` 输入框中，点击 `Sign In`。
5. **确认注销**：
   - 登录成功后，页面会自动跳转到 `Deactivate Account`（注销账户）选项。
   - 网页会询问你注销的原因（可留空）。
   - 点击最下方的蓝色 **「Delete My Account (删除我的账户)」** 按钮。
   - 在弹出的红色警告确认框中，点击 **「Yes, delete my account (是的，确定删除)」**。
6. **成功提示**：页面刷新并显示 “Your account was successfully deleted from our system...”，即代表销号成功。

::: warning ⚠️ 提示：国内网络访问限制
国内网络可能无法直接打开 `my.telegram.org` 页面，需要自备代理网络。如果页面提示登录失败或收不到验证码，请确保输入格式完全正确，或更换浏览器（推荐 Chrome/Edge）并使用隐私模式重试。
:::

## 二、自动销号机制（自我毁灭定时器）

为了减少不活跃账号占用服务器资源，并保障丢失账号的隐私安全，Telegram 默认开启了**自动销号**机制。

### 2.1 如何修改自动销号期限
如果你在设定的一段时间内没有使用过 Telegram（即没有任何一台登录你账号的设备连接过服务器），系统就会自动执行注销清空操作。

- **配置路径**：打开 Telegram 手机端 -> **「设置 (Settings)」** -> **「隐私与安全 (Privacy and Security)」**。
- **配置项**：滚动至底部找到 **「如果离开时间超过 (Delete my account if away for)」**。
- **可选期限**：**1个月、3个月、6个月、1年**。
- **建议**：建议设为 **1年 (12 Months)**，以防止因临时回国、手机卡停机等原因导致短时间内无法登录时，账号被意外注销。

## 三、避坑指南：销号前的两件关键大事

::: danger 🚨 极重要警告：销号的后果
1. **数据无法恢复**：所有的私聊记录、群聊记录、发送的媒体文件、收藏夹 (Saved Messages) 将会被彻底从云端服务器抹除。
2. **群组与频道关系断开**：你将自动退出所有加入的群组与频道。
:::

为了避免不必要的损失，销号前必须做好以下两点：

### 3.1 转移你创建的群组与频道所有权 (Transfer Ownership)
如果你是某些群组或频道的**创建者 (Owner)**，直接销号会导致这些群/频道变成“群主/创建者已注销 (Deleted Account)”。虽然群组能继续存在，但未来你将永久失去修改群设置、增减管理员等最高权限。

**转移所有权步骤**：
1. 进入你想转让所有权的群组或频道。
2. 管理群组/频道 -> **管理员 (Administrators)** -> 点击 **添加管理员 (Add Administrator)**。
3. 选择你信任的接管人（必须已经是群/频道成员），并在下方权限开关中，开启**所有管理权限**。
4. 开启所有权限后，底部会出现一个红色的 **「转让群组所有权 (Transfer Ownership)」** 按钮。
5. 点击该按钮，根据提示输入你的 2FA 两步验证密码，即可瞬间将创建者身份转移给对方。

### 3.2 备份重要数据
如果你有大量重要的对话或资料：
- 请参考 [Telegram 数据导出完全指南](./data-export.html) 教程。
- 使用 **Telegram Desktop (桌面版)** 的数据导出功能，将所有内容备份为 HTML 或 JSON 文件存入电脑。

## 四、销号后常见问题 (FAQ)

### Q1：销号后，我的 Fragment 资产（用户名/888号码 NFT）会消失吗？
**不会。** Fragment 的所有资产都是独立存储在你的 **TON 区块链钱包（如 Tonkeeper）** 中的，而不是存储在 Telegram 账号里。销号只会解除这些 NFT 与该电报号的绑定关系。你可以在注册新电报号后，将 TON 钱包重新关联到新号，并把用户名或号码重新绑定到新账号上。

### Q2：我刚刚销号，可以立刻用同一个手机号重新注册吗？
**可能会有冷却期。** Telegram 官方为了防范恶意注册和垃圾邮件行为，在销号后会执行数天至数周不等的**冷却拦截**。如果你立即注册，系统可能会提示“手机号受限”或“尝试次数过多”。遇到这种情况，只需静置 3-7 天后再重新注册即可。

### Q3：重新注册新账号后，我原来的好友还能看到我吗？
对于你的联系人：
- 原来与你的聊天窗口会显示为 **「Deleted Account」**，且头像变成幽灵图。
- 当你注册新账号后，系统会将你视作一个**全新用户**。你的联系人如果手机通讯录里还保存着你的号码，他们会收到 “XX joined Telegram” 的通知，但他们与你过去的聊天记录已不复存在。
