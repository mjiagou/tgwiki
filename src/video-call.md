# Telegram视频通话与语音聊天完整指南

Telegram提供了完善的音视频通讯功能，包括一对一视频通话、群组语音聊天室和屏幕共享等。本指南将详细介绍这些功能的使用方法。

## 一对一视频/语音通话

### 发起通话

1. 打开与联系人的私聊窗口
2. 点击右上角的 **📞 电话图标**（语音通话）或 **🎥 摄像机图标**（视频通话）
3. 等待对方接听

::: tip 注意事项

- 视频通话需要双方都使用支持视频通话的Telegram版本
- 首次使用可能需要授权摄像头和麦克风权限

:::

### 通话中操作

在通话界面，您可以进行以下操作：

- **静音/取消静音**: 点击麦克风图标
- **开启/关闭摄像头**: 点击摄像机图标
- **切换前后摄像头**: 点击摄像头切换图标（仅移动端）
- **开启扬声器**: 点击扬声器图标
- **结束通话**: 点击红色挂断按钮

## 群组语音聊天室

群组语音聊天室（Voice Chat）允许群组成员进行多人实时语音交流，无需拨打电话。

### 创建语音聊天室

**群主或管理员操作：**

1. 打开群组聊天窗口
2. 点击右上角 **⋮**（更多选项）
3. 选择 **"开始语音聊天"**
4. 设置聊天室名称（可选）
5. 点击 **"开始"**

::: tip

语音聊天室创建后，群组成员会收到通知，可以自愿加入。

:::

### 加入语音聊天室

**普通成员操作：**

1. 点击群组中的语音聊天室通知横幅
2. 或者点击右上角 **🎤 麦克风图标**
3. 选择 **"加入语音聊天"**

### 语音聊天室管理

**管理员可以：**

- **静音/取消静音成员**: 长按成员头像
- **移除成员**: 长按成员头像选择移除
- **结束语音聊天**: 点击右上角结束按钮

## 屏幕共享功能

Telegram支持在视频通话中共享屏幕，方便演示内容。

### 开启屏幕共享

**在视频通话中：**

1. 点击界面底部的 **"共享屏幕"** 按钮
2. 授权屏幕录制权限
3. 选择要共享的屏幕或应用窗口
4. 点击 **"开始直播"**

::: warning 注意

屏幕共享可能会暴露敏感信息，请在安全环境下使用。

:::

### 结束屏幕共享

点击屏幕上的 **"停止共享"** 按钮即可结束。

## 高级功能

### 噪音消除

Telegram内置了AI噪音消除功能，可以过滤背景噪音。

**开启方法：**

1. 在通话界面点击 **⋮**（更多选项）
2. 选择 **"噪音消除"**
3. 选择降噪级别（低/中/高）

### 音频优化

- **语音增强**: 优化人声清晰度
- **音乐模式**: 提升音乐播放质量（部分版本支持）

## 常见问题排查

### 无法发起视频通话

**可能原因：**

1. 网络不稳定
2. 对方版本不支持
3. 权限未授权

**解决方法：**

- 检查网络连接
- 更新Telegram到最新版本
- 在系统设置中授权摄像头和麦克风权限

### 通话质量差

**优化建议：**

1. 切换到更稳定的Wi-Fi网络
2. 关闭其他占用带宽的应用
3. 使用耳机提升音质
4. 开启噪音消除功能

### 屏幕共享无画面

**解决方法：**

1. 检查屏幕录制权限
2. 重启Telegram应用
3. 确保系统版本兼容
